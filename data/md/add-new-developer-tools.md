# 添加新的机场节点到推荐列表

按照以下步骤添加新的机场节点到推荐列表。

## 第一步：添加新的机场

1. 打开网站的后台管理系统
2. 选择"添加新机场"选项
3. 填写机场的相关信息：
   - 机场名称
   - 机场简介
   - 官网地址
   - 支持的协议类型
   - 价格区间
   - 节点分布
   - 测速信息

## 第二步：提交更改

在提交之前，请确保完成以下检查：

- [ ] **名称**：提供准确的机场名称
- [ ] **描述**：清晰说明机场的主要特点和优势
- [ ] **网址**：提供有效的官方网站地址
- [ ] **数据准确性**：确保所有信息真实可靠
- [ ] **可访问性**：确认链接可以正常访问

### 注意事项
- 只收录稳定运营的机场
- 不接受含有欺诈信息的机场
- 确保所有信息真实有效
